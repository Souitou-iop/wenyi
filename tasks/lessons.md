# Lessons

- 进度条验收必须同时验证数值持续推进、阶段文案和持久化断点，不能只确认控件出现或比例变化。
- 外部终止 worker 会绕过 App 的“先标记暂停”路径；停止状态必须在发送终止信号前持久化，退出回调不得覆盖已暂停状态。
- 累计运行时间应按 worker 活跃区间和最后心跳结算，不能用自然时间简单相减，否则暂停和 App 离线时间会被误计入。
- `navigationSplitViewColumnWidth` 在 macOS 上是布局建议而非严格拖动上限；需要硬限制时使用 `NSSplitViewItem` 厚度约束和 delegate 坐标限制，并必须重启到实际新构建后再验收。
